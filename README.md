# Home task for mastercard

## How to build ?

1. Install JDK 11
2. Install maven 3.6
3. `mvn package`

## How to run ?

```shell
java -jar target/homework.jar tasla.csv 2
```

It will print pivot points for file tesla with pivot depth 2
